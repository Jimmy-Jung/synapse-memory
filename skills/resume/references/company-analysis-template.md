# 회사 심층 분석 템플릿 (company-analysis-template)

`/sm:resume` Phase 1 이 생성하는 `Companies/<slug>/analysis.md` 의 골격이다.
**JD 요약에서 그치지 않는다.** 회사 홈페이지·기술블로그·뉴스·IR·평판을 웹 조사해 채운다. 모든 비자명 주장에 출처 URL을 단다. 검증 불가한 수치는 적지 않는다.

`{{...}}` 치환 슬롯, `<!-- -->` 작성 지침.

---

```yaml
---
company_id: {{slug}}
display_name: {{회사명}}
status: target            # target | applied | interview | hired | rejected
country: {{국가}}
size: {{small|medium|large}}
website: {{도메인}}
source_url: {{공고 URL}}
positions:
  - title: {{포지션명}}
    seniority: {{junior|mid|senior}}
    keywords: [{{키워드들}}]
researched: '{{YYYY-MM-DD}}'
author: {{Git 사용자명}}
tags: [node/card]
---
```
