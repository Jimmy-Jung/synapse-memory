# 경력기술서 정본 템플릿 (resume-template)

`/sm:resume` 가 생성하는 **최종 이력서(resume.md)** 의 골격이다.
기준 샘플: `정준영 2026_개선 0514.md` (메가스터디교육 + 수학에심장을달다). 섹션 순서·깊이·정량지표 밀도를 이 수준으로 맞춘다.

규칙은 `style-guide.md` 를 따른다. 아래 `{{...}}` 는 치환 슬롯, `<!-- -->` 는 작성 지침(출력물에 남기지 않음).
